# countdown

**Usage**: `@Eros countdown [method] [arguments]`

**Aliases**: `countdown`, `cd`

**Brief Description**: Displays countdowns related to Kamihime Project in-game events.

It includes special and some regular events.

Available Methods:

* `help`
* `add`
* `remove`
* `del`
* `delete`
* `test`
* `check`
