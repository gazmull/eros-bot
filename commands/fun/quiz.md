# Command: quiz


**Usage**: `@Eros quiz `

**Aliases**: `quiz`, `trivia`

**Brief Description**: Deploys a questionnaire related to Kamhime Project.



## Required Bot Permissions

```
Send Messages
Manage Messages
Add Reactions
Embed Links
```

Gain EXP rewards by using this command!
