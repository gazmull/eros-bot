# Command: level


**Usage**: `@Eros level <method> [arguments]`

**Aliases**: `level`

**Brief Description**: Parent command of leveling system.

Available Methods:

	- `info`

	- `i`

	- `leaderboard`



## Required Bot Permissions

```
Send Messages
Manage Messages
Add Reactions
Embed Links
```

## Examples

```
@Eros level level info @Euni
@Eros level level i @Euni
@Eros level leaderboard 5
```
