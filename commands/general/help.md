# Command: help


**Usage**: `@Eros help [command name]`

**Aliases**: `help`, `commands`

**Brief Description**: Displays available commands or command information



## Required Bot Permissions

```
Send Messages
Manage Messages
Add Reactions
Embed Links
```

## Examples

```
@Eros help 
@Eros help ping
@Eros help info
```
