# Command: leaderboard


**Usage**: `@Eros leaderboard [page number]`

**Aliases**: `leaderboard`, `lb`, `toppeeks`, `top`

**Brief Description**: Displays leaderboard for top views on harem scenes from Kamihime Database.



## Required Bot Permissions

```
Send Messages
Manage Messages
Add Reactions
Embed Links
```

## Examples

```
@Eros leaderboard 
@Eros leaderboard 13
@Eros leaderboard 37
```
