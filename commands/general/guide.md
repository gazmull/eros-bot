# Command: guide


**Usage**: `@Eros guide [page number]`

**Aliases**: `guide`

**Brief Description**: Displays guide for using this bot and other game-related information.



## Required Bot Permissions

```
Send Messages
Manage Messages
Add Reactions
Embed Links
```

## Examples

```
@Eros guide 
@Eros guide 13
@Eros guide 37
```
