# Command: prefix


**Usage**: `@Eros prefix <prefix value>`

**Aliases**: `prefix`

**Brief Description**: Changes this server's prefix.



## Required Bot Permissions

```
Send Messages
Manage Messages
Add Reactions
Embed Links
```

## Required User Permissions

```
Manage Server
```

## Examples

```
@Eros prefix e?
@Eros prefix eros
```
